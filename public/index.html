PlaceHolder Site

This site will generate placeholder images of different sizes and colors. You can access a placeholder image by entering its size in pixels in the URL, like this:

  • /100 - generates a 100x100 image
  • /500 - generates a 500x500 image
  • /700 - generates a 700x700 image

You can also change the background color, width, height, and placeholder text and its color using parameters, like this: Params ==> w=Width, h=Height, c=Background Color, txt=PlaceHolder Text, txtc=TextColor

  • /?w=400&h=200&c=skyblue - generates a 400x200 image with skyblue background
  • /?w=300&h=600&c=skyblue&txt=Hello World&txtc=green.
  • - generates a 300x600 skyblue image with the text "Hello World" in green.
  • if you leave some parameters, The default value will added to them.
  • Provide color in RGB format or name eg: rgb(255, 255, 0) Or Green/yellow etc.
